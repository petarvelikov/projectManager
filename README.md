# Project Manager

Laravel example project.

## How to install

1. Clone or download this repository
2. Copi file .env.example to .env
3. Install dependencies

```
composer install
```
composer install

4.  php artisan key:generate
4. php artisan migrate
5. Start project
```
php artisan serve
```
